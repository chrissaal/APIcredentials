# Campus Digital Credential API Examples

Ejemplos separados por lenguaje e IDP para el contrato Campus Digital Credential API 2.4.0.

## Node.js

- `credential-api-node-azure`: implementación Node.js/TypeScript con Azure Entra ID.
- `credential-api-node-google`: implementación Node.js/TypeScript con Google IDP.

## PHP

- `credential-api-php-azure`: implementación PHP 8.2+ con Azure Entra ID.
- `credential-api-php-google`: implementación PHP 8.2+ con Google IDP.

Cada carpeta incluye su propio README, `.env.example`, validadores de contrato y pruebas.
